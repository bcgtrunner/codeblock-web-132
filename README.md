## Интерпретатор

**Текущий целевой уровень: 🔥Hard🔥**

Реализованы все требования, относящиеся к интерпретатору из первых 2 категорий.
Из категории Hard реализованы:

- Возможность объявлять и работать с функциями.
- Использование сложных вложенных арифметических выражений в логических  выражениях, и в индексах массивов.
- Работа со строками и с символьными типами данных.
- Вещественный тип переменных.
- Конвертация типов переменных.
> Любой элемент, который есть в более “общественно признанных” языках программирования, делающий возможности вашего языка более мощными (Матрицы, структуры данных, ООП, лямбды…)
  - массивы могут быть сколь угодно вложенными, так что любые коллекции принципиально возможны
  - лямбда функции: функции можно сделать безымянными, и стандартный способ объявления функции есть объявление через function expression, не function declaration.
  - помимо этого: у переменных есть области видимости. Есть стэк вызовов.
  - у функций есть сигнатуры с runtime проверкой
  - тип void
- Режим отладки и пошагового исполнения (Debug Mode).
