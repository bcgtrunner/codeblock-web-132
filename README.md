# codeblock web project (WIP)
## Текущий статус: Bubble Sort
На данный момент интерпретатор полностью поддерживает **Тьюринг-полный** набор команд, достаточный для реализации алгоритмов, включая реализованную _сортировку пузырьком_.
### Реализованный функционал (Основа)
- Декларация переменных
- Арифметический вычисления
- Оператор присваивания
- Ветвление (If-Else)
- Цикл (While)
- Массивы: Работа со статическими массивами.
- Логика: операторы AND, OR, NOT и операторы сравнения.
### 🔥 Продвинутые возможности (HARD)
- Поддержка возвращаемых значений (return)
- Вложенность выражений (eval)
- Использование вычислений внутри индексов массивов: Array[getStep(i) + 1]
- Типы данных: Float
